Data Collection and Processing

Biometric Data
Seren collects data from Polar brand sensors worn by users to monitor heart rate and calculate heart rate variability (HRV). This data allows the platform to provide stress-related metrics.

Geolocation and Accelerometer Data
Seren also collects geolocation and accelerometer data from users' phones to support the performance of first responders' duties.

Data Processing
Data flows through the platform in the following path:

  • Sensor to App: The sensor transmits data via Bluetooth to the Seren app.

  • App to Cloud: Data is securely transmitted via HTTPS to AWS servers.

  • Cloud to Database: Data is encrypted, error-checked, and stored in a secure relational database.
